# 🤖 Python Voice Assistant

A simple AI-based Voice Assistant built using Python.  
It can listen to voice commands and perform basic tasks like opening websites, searching the web, and interacting with the system.

---

## 🚀 Features

- 🎤 Voice command recognition  
- 🌐 Open websites (Google, YouTube, etc.)  
- 🔍 Web search automation  
- 🗣️ Text-to-speech responses  
- 💻 Basic system automation  

---

## 🛠️ Technologies Used

- Python 🐍  
- SpeechRecognition  
- Pyttsx3 (Text-to-Speech)  
- PyAudio  
- Webbrowser module

## ⚙️ Installation & Setup

### 1. Clone repository
```bash
git clone https://github.com/asfa-shamim/python-voice-assistant.git
```

### 2. Go to project folder
```bash
cd python-voice-assistant
```

### 3. Install dependencies
```bash
pip install -r requirements.txt
```

### 4. Run project
```bash
python app.py
```

---

## 🎯 How it works

1. Run the program  
2. Speak a command  
3. Assistant listens and responds  
4. Performs tasks like opening websites or searching
